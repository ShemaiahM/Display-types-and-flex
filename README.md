### Simple Gloss over
- flow layout the default settings of the html page on teh browser without Css.
- h1,h2,h3,h4,h5,h6 and p tag are all display block type. They always start on a new line.
- strong/bold, em and a tags are all inline tags. Now these do not start on a new line.
- img tag is both inline block, you can change the width and height of this element (like a block element).

- display flex(css) turns the default into a flexible layout so you can have granular control of elements.
- justify-content changes the placement of content on the horizontal line. If you change the flex direction, the axis changes, but this property always controls the main axis.
- align-items changes the postion of the elements on the vertical axis. By default, this manages vertical placement (top, bottom, center, stretch). It controls the axis perpendicular to the main axis.
